NeuralChat 💬

Messenger-Style Demo Chat Interface

NeuralChat is a demo chat application that simulates a real-time messaging experience similar to modern messaging platforms.

Users can send messages through a chat interface, and the system automatically generates demo replies to simulate a conversation. The responses are not powered by real AI APIs — they are preconfigured for demonstration purposes.

This project is built to showcase chat UI design and frontend interaction using JavaScript.

---

✨ Features

- 💬 Messenger-style chat interface
- ⚡ Automatic demo replies
- 📱 Mobile friendly layout
- 🎨 Clean messaging UI
- ⌨️ Send messages instantly
